## THE BESPOKE BRIEF

REDEFINING E-COM AUTOMATION THROUGH MINIMALIST DESIGN

This app solves three problems of E-COM sellers: Shipping label cropping, Selling price suggestion, and calculating hidden losses per order.

`All three in one.`

## STARLIGHT HEADLINER

![Image showing PDF cropping feature of the software](/docs/headliner-1.png)

### PDF CROPPING
The PDF cropping feature of the software makes it an everyday use software for E-COM sellers. With this feature sellers can crop out the shipping labels from different platform invoices and print & paste them on products.

---

![Image showing Hidden loss calculation system of the software](/docs/headliner-2.png)

### HIDDEN LOSS CALCULATION
The hidden loss calculation system of the software makes this one of the most useful software for E-COM sellers. With this system sellers can calculate hidden losses per order on their products.

---

![Image showing Pricing calculation system of the software](/docs/headliner-3.png)

### PRICE CALCULATION
The price calculation system of this software is built to setup a clean pricing of products making the odds of loss as low as possible. This system allows sellers to calculate the best pricing of their products to avoid losses.

---

![Image showing ROI calculation system of the software](/docs/headliner-4.png)

### ROI CALCULATION
The ROI calculation system of the software allows sellers to calculate weather their ads on specific products are actually making revenue and profit or they're just another reel skipped. This feature gets seller a clean ROI analytics.

---

![Image showing FBA generation system of the software](/docs/headliner-5.png)

### FBA GENERATION
The FBA Generation system of this software is designed to generate a clean, bulk-printable FBA label with barcode for specific products. With this system sellers can paste their credentials and product details on their products.
